Add new source-file, executable, scripts (handle CMake)

HPhi uses CMake for the building system. We have to modify the CMake configuration file when we add new sources, executables, scripts.

New source code

When we add new source code for the HPhi program, we have to add the file-name into the following part of src/CMakeLists.txt

1 set(SOURCES source1.c source2.c ...)

New executable

When we add new executable ("myprog" in this case), we have to add following command in src/CMakeLists.txt.

1 set(SOURCES_MYPROG source1.c source2.c ...)
2 add_executable(myprog ${SOURCES_MYPROG})
3 target_link_libraries(myprog ${LAPACK_LIBRARIES} m)
4 if(MPI_FOUND)
5 target_link_libraries(myprog ${MPI_C_LIBRARIES})
6 endif(MPI_FOUND)
7 install(TARGETS myprog RUNTIME DESTINATION bin)

New script

When we add new script written in python, sh, etc. ("myshell" in this case) into tool/, we have to add the following command in tool/CMakeLists.txt.

1 configure_file(myscript.sh myscript.sh COPYONLY)
2 install(FILES ${CMAKE_CURRENT_BINARY_DIR}/myscript.sh DESTINATION bin
3  PERMISSIONS OWNER_READ OWNER_WRITE OWNER_EXECUTE GROUP_READ GROUP_EXECUTE WORLD_READ WORLD_EXECUTE)
